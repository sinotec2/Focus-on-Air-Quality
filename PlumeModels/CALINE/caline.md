# CALINE3
{: .no_toc }
此處介紹CALINE3模式。

CALINE3是美國環保署公告的[替代模式](https://sinotec2.github.io/Focus-on-Air-Quality/PaperReview/LargeSSPtSrcEIA/1Gaus_Stab/#usepa-scram模式種類架構)，意味一般狀況線源排放的模擬還是應該用AERMOD，而針對近距離路緣側的空品、特殊大氣條件下，也還是可以用CALINE3保守推估的。
