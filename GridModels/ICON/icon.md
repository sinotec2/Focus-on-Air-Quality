# 初始條件之準備
{: .no_toc }

**CMAQ**模式的初始條件有3種給定方式：
1. 由全球模式模擬結果檔案解讀(`REGRID`)
1. 由前一批次**CMAQ**執行成果解讀(`REGRID`)
1. 以一組觀測值或符合化學平衡的模擬結果設定(`PROFILE`)
