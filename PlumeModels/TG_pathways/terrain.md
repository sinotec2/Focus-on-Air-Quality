# 地形檔案之準備
{: .no_toc }

地形檔案因與模擬範圍、接受點解析度等有密切關連，因此通常是煙流模式模擬作業遇到的第一項挑戰。
- 模擬範圍需符合高斯模式均勻大氣的假設，不能超過50Km，不能有山脈阻檔
- 解析度不需太密致濃度差異太小
- 有關AERMOD在臺灣應用的實務討論，可以參考[環工技師會訊11007pp39-55](http://www.tpeea.org.tw/upload/news/files/7eea35bc4c7a4189b42566fffe2f2fee.pdf)
