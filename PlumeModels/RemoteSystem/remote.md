# 遠端執行系統
{: .no_toc }

利用遠端伺服器進行模式檔案的準備、主程式計算、後處理、繪圖等作業有重要的利基：
- 避免程式IO、檔案規格等資訊系統問題造成的屏障
- 可以有符合模式技術規範的設定方式與合理的預期結果
- 避免大量使用本地計算資源(記憶體、計算核心、儲存檔案)
