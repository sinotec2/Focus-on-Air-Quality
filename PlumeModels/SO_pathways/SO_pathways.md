# SO Pathways
{: .no_toc }
- 此處介紹ISC/AERMOD有關污染源、排放量等設定方式。
- SO 路徑這一段落雖然不是在模式控制檔的第一個位置，卻是使用者需要最先掌握的資訊：
  - 污染源是什麼
  - 座落何處、可能影響範圍、附近擴散情形、地形地物、有哪些敏感點?
  - 污染排放量是多少?時間特性為何?
